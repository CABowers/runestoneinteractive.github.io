Approximating pi
================

.. activecode:: pi_example1

   terms=input("How many terms do you wish?")
   pi = 0.0
   den = 1
   for counter in range(terms):
       nextterm = (4.0/den)*(-1)**counter
       pi = pi + nextterm
       den = den + 2
   print pi


   def leibniz(terms):
       acc = 0.0
       num = 4.0
       den = 1
       for term in range(terms):
           nextterm = num/den * (-1)**term
           acc = acc + nextterm

           den = den + 2

       return acc

Try running the code above with different values for the
number of terms.  How many terms do you need in order to get
3.14159 for the value of pi?

Getting Started

Get the Template Project
The quickest way to get up and running is to install the runestone tools with pip: pip install runestone
Create a Folder for your project
mkdir myproj You can initialize a template project for yourself by running runestone init inside the project folder. It will ask you some questions. You can either take the defaults or customize your project for your own environment.
Edit
The template project has an index.rst file ready to edit. You can simply start with the index or overview file and build up from there.
Build
The runestone build command will create a nice looking html file for you in the build directory under the template project. Once you have built you can preview that file directly in your browser.
Deploy
You can serve your project right from your own computer by running runestone serve Or you can copy the folder in the build subdirectory created by the build command anywhere that will serve a static website, including a site that supports github pages.
